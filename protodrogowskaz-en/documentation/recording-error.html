Getting Started
Create a component
Components are groups of steps that are shared across multiple tests. If you change it in one test it will be changed in all the tests that use this component.
For example all your tests start with opening a homepage URL. You don't need to record this each time. You can record it once, make it a component and use it in all your tests.
Tip: after your tests use components, you can use "New test from here" on tests graph which will create a new test based on components and groups from existing tests. You can also make components from this menu!
